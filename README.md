# Final Project - Grocery Notebook App



## Description

- Add grocery items.
- Delete grocery items.
- Notebook-style design with a lined paper effect.

## Getting Started

### Dependencies

```
pip install -r requirements.txt
```

### Executing program

```
python run.py
```

## Help

This will create a Notebook to add a list of groceries to be bought.
